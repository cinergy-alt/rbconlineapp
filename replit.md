# RBC Online Banking Clone

## Overview
A React-based clone of the RBC (Royal Bank of Canada) Online Banking interface. This is a frontend-only demo application with a login page and account dashboard.

## Tech Stack
- React 19
- Vite 7
- Tailwind CSS 4
- Lucide React (icons)

## Project Structure
```
/
├── index.html          # HTML entry point
├── vite.config.js      # Vite configuration (port 5000)
├── postcss.config.js   # PostCSS with Tailwind
├── src/
│   ├── main.jsx        # React entry point
│   ├── App.jsx         # Main application component
│   └── index.css       # Tailwind imports
└── package.json        # Dependencies and scripts
```

## Running the Application
- Development: `npm run dev` (runs on port 5000)
- Build: `npm run build`
- Preview: `npm run preview`
